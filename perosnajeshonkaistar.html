Sobre esta Página

Esta página fue creada como un espacio visual y temático para compartir imágenes, videos y detalles interesantes del universo de Genshin Impact y Honkai: Star Rail. En ella se destacan personajes como Yukong, Jade y Fu Xuan, con galerías diseñadas al estilo Pinterest para una navegación atractiva y ordenada.

El objetivo principal es ofrecer una experiencia inmersiva y estética para los fans de ambos juegos, combinando elementos visuales, contenido informativo y recomendaciones útiles para jugadores nuevos y veteranos.

🔧 Tecnologías utilizadas: HTML5, CSS3, diseño responsive y embebido de multimedia (YouTube).

🌐 Esta web no tiene fines comerciales y ha sido desarrollada como proyecto educativo/personal para explorar la creatividad digital y el diseño web moderno.
